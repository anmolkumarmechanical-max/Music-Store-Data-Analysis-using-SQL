📌 Project Overview

This project focuses on analyzing a fictional music store database using SQL to uncover valuable business insights related to customers, sales, and music trends.

It demonstrates the use of:

SQL Joins
Aggregations (GROUP BY, SUM, COUNT)
Filtering & Sorting
Common Table Expressions (CTEs)
Window Functions

The project is designed to simulate real-world data analysis tasks performed by data analysts in the music and entertainment industry.

🎯 Objectives
Analyze customer purchasing behavior
Identify top-performing artists and genres
Discover high-revenue locations
Provide data-driven business recommendations

| Table Name   | Description                                     |
| ------------ | ----------------------------------------------- |
| employee     | Employee details including job title and levels |
| customer     | Customer information (name, email, country)     |
| invoice      | Billing details and total purchases             |
| invoice_line | Line-level transaction details                  |
| track        | Track information including genre and duration  |
| album        | Album details                                   |
| artist       | Artist information                              |
| genre        | Genre classification                            |
